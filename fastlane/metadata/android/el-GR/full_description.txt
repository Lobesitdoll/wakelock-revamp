Το Wake Lock σας δίνει πρόσβαση στην Διαχείριση Ενέργειας και Wifi του Android.
Μπορείτε να το χρησιμοποιήσετε σε οποιοδήποτε τηλέφωνο ή tablet με λογισμικό Android.

Παραδείγματα του τι μπορεί να κάνει για σας η εφαρμογή:
• Να αναγκάσει το PowerManager να κρατήσει την οθόνη ενεργοποιημένη
• Να κάνει τον Επεξεργαστή να συνεχίσει να λειτουργεί σε κατάσταση αναμονής
• Να βεβαιώσει ότι η σύνδεση Wi-Fi θα συνεχίσει να λειτουργεί με πλήρη απόδοση
• Να διατηρήσει την οθόνη σε πλήρη φωτεινότητα ή σε κατάσταση χαμηλού φωτισμού κατά τη διάρκεια προβολής των ταινιών
• Παρακάμψει τα μέτρα Εξοικονόμησης Ενέργειας εάν προκαλούν ζητήματα

Αυτό είναι η εκσυγχρονισμένη έκδοση της εφαρμογής μου «WakeLock - PowerManager».

Τα δικαιώματα χρησιμοποιούνται για:
• WAKE_LOCK, προφανώς για να επιτρέπεται να αποκτήσει wakelocks.
• RECEIVE_BOOT_COMPLETED, για να ξεκινήσει η εφαρμογή, μετά την επανεκκίνηση της συσκευής.
• READ_PHONE_STATE, για να ενεργεί κατά την έναρξη/τερματισμό των κλήσεων, επιτρέποντας στην εφαρμογή να αποκτήσει ένα κλείδωμα μόνο για τη διάρκεια της κλήσης.
• INTERNET, για την αυτόματη παρακολούθηση σφαλμάτων. Υπάρχει μια επιλογή για να το απενεργοποιήσετε αυτό, αλλά γιατί να το κάνετε;; 🙁
